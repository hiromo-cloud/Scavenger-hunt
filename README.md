# 🎒 お仕度ハント (Scavenger Hunt)

お子様の毎朝の「お仕度」を、ワクワクする宝探しゲームに変える React アプリケーションです。

## 🌟 主な機能

- **ルーレット形式のお仕度**: 次に何を用意するか、キャラクターが楽しくガイドします。
- **カスタマイズ**: お子様の写真や好きなキャラクターを応援団に設定可能。
- **スケジュール管理**: 曜日やシーン（保育園・学校・おでかけ）に合わせて持ち物を設定。
- **ハント記録**: 準備にかかった時間をグラフで確認し、成長を実感。

## 🛠 技術スタック

- **Frontend**: React (Hooks, Context API)
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Backend**: Firebase (Authentication, Firestore)
- **Icons**: Lucide React
- **Fonts**: M PLUS Rounded 1c
